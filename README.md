# Home Cooking Helper

# Overview
This web application will serve as a comprehensive platform for home cooking, offering users guidance on everything they need to know to prepare food in the comfort of their own homes. The application enables users to input the ingredients they have in their kitchen and receive recommendations for recipes that can be made with them. A multitude of customizable filters allows users to refine their search based on dietary restrictions, cuisine type, nutritional composition, flavor profile, dish complexity, and whether additional ingredients can be added.

# Features
## Finding Recipes Based on Ingredients
Users can input a list of ingredients they have on hand and receive recipe recommendations from the database. They can refine their search using customizable filters, including:

- Average rating
- Cuisine type
- Complexity
- Preparation time
- Dietary restrictions
- User Recipe Reviews and Ratings
- Users can rate recipes they have tried and view reviews from other users in the social tab. This feature helps users make informed decisions based on community feedback.

## Saved Recipes
Users can create accounts to save specific recipes for future use, making meal planning more convenient.

## Upload Recipes
Users can upload their own recipes for others to see and interact with in the same ways as the default recipes. This encourages community engagement and recipe sharing.

## Rank Recipes
Users are able to rank the recipes they have tried, creating a sorted list of past recipes they enjoyed or those they would rather not try again.
